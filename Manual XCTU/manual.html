Manual de configuración - Modulo Wifly

1

Conexión e instalación

Para comenzar, se inicia con la conexión del Módulo wifly conectandolo via USB al ordenador. El modulo wifly será el encargado de interconectar el cliente android que contiene la aplicación, con el robot a controlar.

Nota: El modulo wifly requiere la instalación del driver para funcionar correctamente

2

XCTU-Terminal

La terminal es la herramienta que se utilizar para configuar el módulo, para conectarse satisfactoriamente se debe configurar los siguientes parametros como se muestran en la imagen:

  • Puero COM
  • BAUD
Configuración XCTU

Nota: El modulo wifly puede estar conectado en otro puerto COM

4

Configuración IP

Esta configuración parametriza toda la capa de red en el módulo wifly, como la dirección ip, mascara puerto,

Ingresar los siguientes comandos:

  1. set ip address 192.168.43.XX (dirección ip de la red)
  2. set ip port 2000
  3. set ip netmask 255.255.255.0
  4. set ip dhcp 0
  5. set ip protocol 2
  6. save
  7. reboot
3

Configuración WLAN

Esta configuración parametriza toda la conexión via inalámbrica en el módulo wifly, como el nombre de la red y su password, se comienza por usar la pestaña Terminal del programa X-CTU, para iniciar la terminal presionar repetidamente el caracter $ simbolo de dolar.

Ingresar los siguientes comandos:

  1. factory RESET
  2. set wlan ssid (nombre de la zona wifi)
  3. set wlan phrase (password de la zona wifi)
  4. set wlan auth (auth numero) (4 para wpa2-psk, depende del tipo de seguridad)
  5. set wlan channel 0
  6. set wlan join 1
  7. save
