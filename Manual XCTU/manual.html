Manual de configuración - Modulo Wifly

1

Conexión e instalación

Para comenzar, es necesario conectar el Módulo wifly con el ordenador vía USB. El módulo wifly será el encargado de la comunicación entre el cliente android (que contiene la aplicación) y el robot a controlar.


Módulo wifly conectado al ordenador

Nota: El módulo wifly requiere la instalación del driver FT232 para funcionar correctamente.

2

La terminal es la herramienta que se utilizará para configuar el módulo. Para conectarse satisfactoriamente se deben configurar como se muestra en la imagen los siguientes parámetros:

  • Puerto COM ?
  • BAUD ?
Configuración XCTU

Nota: El módulo wifly puede estar conectado en otro puerto COM.

4

Configuración IP

Esta configuración parametriza toda la capa de red y de transporte en el módulo wifly, estableciendo atributos como la dirección IP, máscara, puerto, etc.

Ingresar los siguientes comandos:

  1. set ip address 192.168.43.XX ?
  2. set ip port 2000 ?
  3. set ip netmask 255.255.255.0 ?
  4. set ip dhcp 0 ?
  5. set ip protocol 2 ?
  6. save ?
  7. reboot ?

Módulo wifly, configuración IP, reinicio y conexión a la red
3

Configuración WLAN

Esta configuración parametriza toda la conexión inalámbrica en el módulo wifly, estableciendo atributos como el nombre de la red y su password. Para comenzar, se debe seleccionar la pestaña Terminal del programa X-CTU. Para iniciar la terminal, presionar repetidamente el caracter $ (símbolo de dólar).

Ingresar los siguientes comandos:

  1. factory RESET ?
  2. set wlan ssid XX ?
  3. set wlan phrase XX ?
  4. set wlan auth X ?
  5. set wlan channel 0
  6. set wlan join 1 ?
  7. save ?

Módulo wifly, configuración WLAN
5

Probar resultados

Para verificar que se haya configurado correctamente el módulo, se debe conectar el dispositivo android y el módulo wifly a la misma red, luego correr la aplicación que se utilizará para comunicarse con el robot y finalmente presionar el botón "Show Hex" de X-CTU.
